Process


Moodboards

For this professional portfolio, I wanted it to reflect my program and profession as a software developer. I decided to have it look like an operating system with a navbar similar to the Windows taskbar and a desktop as the landing page with different icons that lead to the different sections. The navbar would act as another way to navigate the website and also include a functioning clock using JavaScript. I referenced a few different operating systems such as Windows, Ubuntu, and Fedora.



Color Palettes

The colors I used reflected every programmers favourite color scheme, the dark mode, as well as some of my favourite colors.




Type Studies

I chose to use a simple font for paragraphs for readability and a monospace font that looks like code for headers.

Ubuntu Mono



JavaScript Interaction

JavaScript was used for the functioning clock on the navbar. It retrieves the year, month, day, hour, and minute and applies it to the clock and date elements in the navbar.



Home Page

Apart from the navbar that persists in all pages, the desktop home page also consists of seven icons, one for each page in the website, and a welcome icon that toggles a modal containing a welcome message. Much like other operating systems, the desktop has a logo of my fictional operating system designed by me. Furthermore, I wanted the landing page to have some kind of animation to give it a bit of movement. As a result, I implemented an animation on load which fades in the logo, then fades in the icons. Each of the icons also has a hover animation where it is enlarged when hovered over.



About Me Page

The about me section contains a brief paragraph about me, some of my technical skills, as well as icons that lead to my resume and the experience / project pages. To keep the design consistent, these icons also have the same animations as the desktop icons.



Experience Page

In this page, I wanted to showcase my job experience as well as some of the languages and technologies used for each of them. There is also a brief paragraph that describes some of my duties for each job. Each of the technologies showcased has an icon that will link to the corresponding websites when clicked on. Each job section also has a small collapsable section at the bottom that reveals some personal notes and reflections for each of the positions, giving the page a bit of a personal touch and allows the reader to learn a bit more about me.



Projects Page

For the experience page, I chose to use an accordian as it easily lists out all of my projects and allows the reader to click and expand different projects for more details. Each of the projects have either images or videos to showcase its functionality or architecture with the exception of the game Cupid's Arrow. It has an icon that leads to the html file for the actual game. Again, the design for the icon is consistent with the dektop icons.



YouTube Page

This page showcases my YouTube channel where I share song mashups and remixes I make. It also showcases a few mashup albums I've made and includes the cover, tracklist, and a brief description of what they are. This page also has an icon that links to my YouTube channel, and once again, the icon has a similar design and the same animation as the desktop icons.



Contact Page

The contact page was kept as simple as possible, making it as easy to reach me as possible. It includes my email and LinkedIn profile as well a form to send me a message.
